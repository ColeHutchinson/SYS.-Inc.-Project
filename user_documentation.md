**Music Catalog**

User Guide

*Browse, search, and manage your music collection*

**Getting Started**

**Logging In**

When you launch the application, you\'ll be greeted by the Login screen.
Enter your username and password, then click Log In.

  --------- -------------------------------------------------------------
  **TIP**   If you\'re using the application for the first time, you can
            log in with the username demo and password demo.

  --------- -------------------------------------------------------------

**Registering a New Account**

If you don\'t have an account yet, click Register on the login screen
and fill in your desired username, password, and email address.

**Browsing Your Music Catalog**

After logging in, the main catalog window opens. You\'ll see a table of
songs with the following columns:

  ---------------------- ------------------------------------------------
  **Column**             **What It Shows**

  Title                  The name of the song

  Artist                 The performing artist or band

  Album                  The album the song belongs to

  Duration               Length of the song (minutes:seconds)

  Genre                  Musical genre (e.g. Rock, Jazz, Pop)

  Year                   Release year of the song

  Date Added             When the song was added to the catalog
  ---------------------- ------------------------------------------------

The status bar at the bottom of the window always shows the total number
of songs and how many are currently visible after any filters are
applied.

**Finding Songs**

**Live Search**

Type anything into the Search box at the top of the window. The catalog
updates instantly as you type, matching your query against song titles,
artists, albums, and genres.

  --------- -------------------------------------------------------------
  **TIP**   You don\'t need to press Enter --- results appear as you
            type. Clear the search box to return to the full catalog.

  --------- -------------------------------------------------------------

**Filtering by Genre**

Use the Genre dropdown to show only songs from a specific genre. Select
All Genres from the dropdown to remove the filter and show everything.

**Combining Search and Genre Filter**

Search and genre filtering work together. For example, you can filter to
Jazz and then search for a specific artist within that genre.

**Sorting the Catalog**

Click any column header to sort the catalog by that column. Click the
same header again to reverse the sort order (ascending to descending, or
vice versa).

You can sort by any of the following columns:

-   Title

-   Artist

-   Album

-   Duration

-   Genre

-   Year

-   Date Added

**Managing Songs**

**Adding a New Song**

1.  Click the Add Song button in the toolbar.

2.  Fill in the song details in the form that appears (Title and Artist
    are required).

3.  Click Save to add the song to the catalog.

**Editing an Existing Song**

There are two ways to open the edit form for a song:

-   Double-click any row in the catalog table.

-   Right-click a row and select Edit Song from the context menu.

Make your changes in the form and click Save. Click Cancel to discard
any changes.

**Deleting a Song**

4.  Right-click the song you want to remove.

5.  Select Delete Song from the context menu.

6.  Confirm the deletion when prompted.

  ---------- -------------------------------------------------------------
  **NOTE**   Deleting a song is permanent and cannot be undone. Make sure
             you\'ve selected the correct song before confirming.

  ---------- -------------------------------------------------------------

**Quick Reference**

  -------------------------- --------------------------------------------
  **Action**                 **How To Do It**

  Search for songs           Type in the Search box --- results update
                             instantly

  Filter by genre            Select a genre from the Genre dropdown

  Clear filters              Empty the search box and select All Genres

  Sort by a column           Click the column header; click again to
                             reverse

  Add a song                 Click Add Song in the toolbar

  Edit a song                Double-click the song row, or right-click →
                             Edit Song

  Delete a song              Right-click the song row → Delete Song

  See total / filtered count Check the status bar at the bottom of the
                             window
  -------------------------- --------------------------------------------

**Troubleshooting**

  ---------------------- ------------------------------------------------
  **Problem**            **What to Try**

  I can\'t log in        Check your username and password. Passwords are
                         case-sensitive. Use demo / demo if you just
                         installed the app.

  No songs appear        Check whether a genre filter or search term is
                         active. Clear both and try again.

  My changes didn\'t     Make sure you clicked Save (not Cancel) in the
  save                   edit form.

  The app won\'t open    Contact your system administrator to verify the
                         application is correctly installed.
  ---------------------- ------------------------------------------------
